# CZ: ENGETO.CZ - #2 Domácí úkol - Responzivní životopis

## Zadání úkolu

### Bonusový úkol č. 2 (nepovinný)

Vytvořte produktovou kartu na libovolní produkt dle vlastního návrhu. Karta může vycházet z návrhu v bonusovém úkolu nebo projektu v lekci. Účelem úkolu je procvičit si probranou látku bez explicitního zadáni.<br>
Úkol můžete dle návodu v linku nahrát na Github a odkaz poslat lektorům do soukromé zprávy na Discordu.<br>

### Vlastnosti projektu

✅ <b>Responzivní design</b> - změna stylů se projeví při okně větší/menší než 968px 📲 / 🖥️<br>
✅ Tlačítko v pravém horním rohu (🌔 měsíček/ 💡žárovka ) pro <b>změnu režimu</b> (<b>tmavý/zelený</b> a <b>modrý/světlý</b>)<br>
✅ Tlačítko pro MUTE/VOLUME 🎶 <b>PŘEHRÁNÍ HUDBY</b> uložené ve složce "assets/mp3/holly_dazedd.mp3" (hraje dokola)<br>
✅ Při kliknutí na <b>tlačítko košíku 🧺</b> u každé karty se zobrazí hláška pro uživatele, že nejde o <b>opravdový eshop</b><br>
✅ Jednotlivé karty jsou <b>generovány pomocí Javascriptu</b> z pole "products" umístěným v souboru <b>"main.js"</b><br>
✅ Na kartě na pozici <b>cena</b> jsou tisíce <b>odděleny mezerou</b><br>
